# 更新日志

所有对 "Coder Theme" 扩展的重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/spec/v2.0.0.html)。

## [1.1.3] - 2026-05-05

### 优化
- 统一 README 文档与主题文件的颜色配置
- 优化语法高亮颜色一致性
- 调整注释颜色对比度，提高可读性
- 优化变量和属性颜色区分度
- 统一所有主题的错误和警告颜色

## [1.1.2] - 2026-04-21

### 修复
- 修复四个主题的引用窗口边界模糊问题，提高视觉清晰度
- 统一引用窗口（peekView、editorWidget、widget）边界颜色与主题强调色一致
- 统一 quickInput 和 commandCenter 边框配色与主题强调色一致

### 优化
- 优化 peekView 标题区域背景色，增强层次感
- 优化 quickInputTitle 背景色，与主题背景协调
- 统一所有主题的 widget 边框样式

## [1.1.1] - 2026-04-14

### 优化
- 优化 Coder Dark Sakura 主题：将主色从青色统一为樱花粉色 #ff7aad
- 优化 Coder Light Sakura 主题：将主色从蓝色统一为樱花粉色 #e91e63
- 更新 Terminal ANSI 配色以匹配各自主題色
- 更新 Charts 配色以匹配各自主題色
- 统一 tokenColors 中的函数、关键字、support 等颜色与主题色一致

## [1.1.0] - 2026-04-11

### 新增
- 新增 Coder Dark Sakura 主题 - 樱花粉色强调色的深色主题
- 新增 Coder Light Sakura 主题 - 樱花粉色强调色的浅色主题
- 两种新主题采用温馨的樱花粉配色方案，提供独特的视觉体验

## [1.0.3] - 2026-04-09

### 修复
- 统一深色和浅色主题的 token 样式配置
- 补全浅色主题缺失的调试相关 token 样式（info、warn、error、debug）

## [1.0.2] - 2026-04-07

### 新增
- 初始版本，包含 Coder Dark 主题
- 初始版本，包含 Coder Light 主题
- 多编程语言的全面语法高亮
- 可自定义的主题强调色（蓝色主题）
- 终端颜色自定义
- 工作台界面主题定制

## [1.0.0] - 2026-03-28

### 新增
- 初始版本发布
- 深色主题变体（Coder Dark）
- 浅色主题变体（Coder Light）
- 完整的工作台颜色自定义
- 编辑器语法高亮
- 终端 ANSI 颜色
- 状态栏样式
- 活动栏样式
- 侧边栏和面板主题
