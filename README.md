## ✨ 功能特性

- **四种主题变体**：
  - 🌙 **Coder Dark** - 精心打造的深色主题
  - ☀️ **Coder Light** - 简洁专业的浅色主题  
  - 🌸 **Coder Dark Sakura** - 樱花粉色强调色的深色主题
  - 🌸 **Coder Light Sakura** - 樱花粉色强调色的浅色主题
- **护眼配色**：针对长时间编码优化的调色板
- **语法高亮**：支持多种语言的全面标记颜色
- **现代 UI**：提供蓝色和樱花粉两种强调色方案

## 🚀 安装

### 通过 VS Code 市场安装

1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展面板
3. 搜索 "Coder Theme"
4. 点击 **安装**

### 通过命令行安装

```bash
code --install-extension Mosaulse.coder-theme
```

## ⚙️ 自定义设置

您可以通过在 `settings.json` 中添加以下设置来进一步自定义主题：

```json
{
  "workbench.colorCustomizations": {
    "[Coder Dark]": {
      // 为 Coder Dark 添加自定义颜色
    },
    "[Coder Light]": {
      // 为 Coder Light 添加自定义颜色
    },
    "[Coder Dark Sakura]": {
      // 为 Coder Dark Sakura 添加自定义颜色
    },
    "[Coder Light Sakura]": {
      // 为 Coder Light Sakura 添加自定义颜色
    }
  }
}
```

## 🎯 推荐配置

为了获得最佳体验，建议配合以下设置使用：

```json
{
  "editor.fontFamily": "Maple Mono NF",
  "editor.fontSize": 14,
  "editor.lineHeight": 1.6,
  "editor.semanticHighlighting.enabled": true,
  "editor.bracketPairColorization.enabled": true
}
```
